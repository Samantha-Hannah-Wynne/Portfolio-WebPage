Continuous Assessment Project

Image credit: Pexels

As the final assignment for my Introduction to Web Development subject, I created a toy store website TinkerTown Toys with a princess motif. This work demonstrates my proficiency with HTML5, CSS3, and JavaScript, with particular focus on user-friendly navigation and adaptive layout. This site has a multi-page design that includes a contact form with JavaScript-based validation, a product/services page with pictures and descriptions of toys, and a homepage. To improve the user experience, I also added interactive elements like product filtering and alert pop-ups. I've also added other features like:

  • Theme toggle (light/dark mode)
  • Language translation (English, French, Tamil)
  • Form validation with JavaScript
